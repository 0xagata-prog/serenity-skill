## 安装方法

### Claude Project（推荐）

1. 打开 Claude → 左侧 **Projects** → **New Project**
2. 点 **Instructions**
3. 复制 [SKILL.md](./SKILL.md) 全部内容
4. 粘贴进去，保存
5. 在这个 Project 里对话即可

### ChatGPT Custom GPT

1. 打开 ChatGPT → **Explore GPTs** → **Create**
2. 切到 **Configure** 标签
3. 把 [SKILL.md](./SKILL.md) 内容粘贴进 **Instructions**
4. 保存
